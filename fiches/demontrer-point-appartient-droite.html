Démontrer qu'un point appartient à une droite à l'aide de la réprésentation paramétrique

On cherche à savoir si le point F appartient à la droite (AB) avec :

  • F (1 ; 0 ; -5)

F appartient à (AB) s'il existe un réel t tel que

{
1 = 3t + 4
0 = -t + 1 , t ∈ ℝ
-5 = -3t + 2

On résout le système suivant, ce qui nous donne :

{
t = -1
t = 1
t = 7/3

Le système n'est pas compatible. F n'appartient donc pas à la droite (AB)
